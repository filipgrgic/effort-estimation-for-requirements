Online Library System

The system shall allow users to create an account using their email address and a password.

Users shall be able to log in and log out of the system.

The system shall allow users to search for books by title, author, or ISBN.

Users shall be able to borrow available books and return borrowed books.

The system shall send an email notification to the user when a borrowed book is overdue.

The system shall display a list of currently borrowed books for each user.

The system shall support at least 1000 concurrent users without performance degradation.

Search results should be displayed within 2 seconds under normal operating conditions.

User passwords must be stored using secure hashing.

The system must comply with the GDPR data protection regulations.

The application must run on Linux servers.

The system shall use PostgreSQL as the database.
